== Description ==

Social Media Slider rotates social media icons for twitter, facebook, youtube, linkedin and rss in any widget. It also can rotate two custom html codes. This is a standalone widget version
of the Social Media Slider that is integrated with the Socrates Theme - http://www.socratestheme.com

== Installation ==

1. Upload and Activate Plugin
2. Goto Appearance > Widgets
3. Drag Social Media Slider widget to any widget location
4. Enter widget title if desired
5. Enter social media profile URLs (anything blank won't rotate)
6. Enter custom html if desired
